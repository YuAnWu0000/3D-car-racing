# 3D賽車遊戲 <br />
## 遊戲說明: <br />
方向鍵控制賽車 空白鍵氮氣加速 <br />
賽道中設置九個檢查點 方便使用者確認行駛方向正確 <br />
時間bar與儀表板的設置增添遊戲性 <br />
行駛一圈後遊戲結束 顯示花費時間 <br />
